# print-a-heart-in-python
print a heart in python

```python
heart = [    "    * * * *       * * * *",    "  *         *   *         *",    " *           *           *",    " *                         *",    "  *                       *",    "    *                   *",    "      *               *",    "        *           *",    "          *       *",    "            *   *",    "              *",    "          "+ "Khanh Ly".center(25)]

for line in heart:
    print(line)
```
